# Greenden Website

GreenDen is a simple and responsive plant-themed website built using HTML, CSS, and Tailwind CSS. The project focuses on creating a clean, modern, and user-friendly interface for showcasing plants and gardening products.

## Features

* Responsive design for mobile and desktop
* Clean and modern UI
* Plant product showcase
* Navigation bar and hero section
* Product cards and categories
* Contact section
* Built with Tailwind CSS utility classes

## Technologies Used

* HTML5
* CSS3
* Tailwind CSS

## Purpose

This project was created to practice front-end web development concepts, improve responsive web design skills, and gain hands-on experience with Tailwind CSS.
